Diferencias entre Java y JavaScript

La primera gran diferencia que conseguimos entre ambos lenguajes es que Java es compilado y JavaScript es interpretado

  2. Fases de depuracion

  4. Versatilidad

Diferencias

  • Los programas hechos en Javascript son archivos de texto que pueden ser leídos tanto por las computadoras como por las personas, en tanto que los programas en Java luego de ser escritos deben ser compilados.
  • En Java la depuración se realiza en dos fases, mientras que en Javascript se realiza en una sola fase
  • Java es un lenguaje de programación orientado a objetos, mientras que Javascript se basa en prototipos
  • Java es un lenguaje fuertemente tipado que no permite cambios o violaciones en los tipos de datos definidos. En Javascript, por se un lenguaje de tipado débil, una variable puede contener diversos tipos de datos, lo que lo hace un lenguaje muy flexible
